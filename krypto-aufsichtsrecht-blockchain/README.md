# krypto-aufsichtsrecht-blockchain

> Dieses Plugin ersetzt keine Rechtsberatung nach dem
> Rechtsdienstleistungsgesetz (RDG) und keine aufsichtsrechtliche
> Einzelfallauskunft der BaFin. Die Ausgaben sind Arbeitshilfen ohne
> Gewaehr. Im Einzelfall ist eine Rechtsanwaeltin oder ein Rechtsanwalt zu
> konsultieren.

Plugin fuer angrenzendes Aufsichts- und Blockchainrecht: MiCAR-
Token-Klassifizierung, BaFin-Erlaubnispflicht, AML/KYC, Travel Rule,
DAO-Rechtsformfragen, Stablecoin-Emittentenpflichten, Smart-Contract-
Haftung und Custody-/Verwahrungsrisiken.

## Skills

| Skill | Thema |
|---|---|
| `token-klassifizierung-mica` | Token-Klassifizierung nach MiCAR |
| `bafin-erlaubnispflicht-kryptowerte` | Erlaubnispflicht fuer Kryptowertedienstleistungen |
| `aml-kyc-kryptodienstleister` | AML/KYC-Pflichten nach GwG |
| `smart-contract-haftung-zivilrecht` | Zivilrechtliche Haftung bei Smart Contracts |
| `dao-rechtsform-einordnung` | Gesellschaftsrechtliche Einordnung von DAOs |
| `stablecoin-emittenten-pflichten-mica` | Pflichten von Stablecoin-Emittenten |
| `travel-rule-kryptotransfers` | Travel Rule bei Kryptowerteuebertragungen |
| `wallet-verwahrung-haftungsfragen` | Haftung und Insolvenzrisiko bei Verwahrung |
