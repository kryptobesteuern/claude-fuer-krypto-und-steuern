# krypto-steuern-de

> Dieses Plugin ersetzt keine Steuerberatung nach Paragraf 3 StBerG und
> keine Rechtsberatung nach dem Rechtsdienstleistungsgesetz (RDG). Die
> Ausgaben sind Arbeitshilfen ohne Gewaehr. Im Einzelfall ist eine
> Steuerberaterin, ein Steuerberater, eine Rechtsanwaeltin oder ein
> Rechtsanwalt zu konsultieren.

Plugin fuer die ertragsteuerliche und umsatzsteuerliche Behandlung von
Kryptowerten in Deutschland: Kauf/Verkauf, Mining, Staking, DeFi, Airdrops/
Hardforks, NFTs, Bewertungsmethoden, Verlustverrechnung und Bilanzierung im
Betriebsvermoegen.
